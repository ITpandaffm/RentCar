# ios租车程序

基于苹果提供的Mapkit所实现的一个lbs小app。

以下为演示demo：

<img src="RentCarDemo.gif" width="400" height="800">

实现功能：

* 首次加载询问获取使用用户定位权限
* 定位成功后跳到用户当前位置
* 自定义放大缩小键
* 自定义大头针
* 按照坐标，在地图上显示标注并分组（附近车辆）
* 点击大头针，显示车辆相应信息
* 等等
